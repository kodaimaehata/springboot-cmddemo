# Getting Started

* Install Java11 and Gradle
* Checkout Repository
* Build and Test  
  Execute command on top of repository.  
    ```gradle build```
* Prepare DBMS environment
* Update application.properties for DBMS
* Execute application  
Execute command on top of repository.  
```gradle run```
